# 📚 CET-4 词汇通

> 一个专为大学英语四级考试（CET-4）打造的静态词汇学习网站，收录 **4252+** 个四级核心词汇，支持搜索、排序、同义词分组、双模式测验和 Canvas 动态背景。

---

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 📖 **词汇学习** | 4252+ 四级词汇，含音标、释义、例句、频率等级 |
| 🔍 **智能搜索** | 支持按单词、释义、音标、例句全文检索 |
| 📊 **多维排序** | 字母顺序（正/反）、频率（高→低 / 低→高） |
| 🏷️ **频率筛选** | 5 级频率标注，快速定位高频核心词 |
| 🔗 **同义词组** | 基于语义相似度的同义词分组，点击可查看组内全部词汇 |
| 🎯 **双模式测验** | 看英文选中文 / 看中文选英文，同义词干扰项生成 |
| 🌠 **动态背景** | Canvas 星空流星动画背景，优雅不干扰学习 |
| 🍀 **四级祝福** | 考场注意事项 + 备考格言 |
| 📱 **响应式设计** | 完美适配桌面端、平板和手机 |

## 🚀 本地使用

本项目为纯静态网站，无需构建工具，直接用浏览器打开即可：

```bash
# 方式一：直接打开
双击 index.html

# 方式二：本地服务器（推荐）
cd cet4-vocab
npx serve .
# 或
python3 -m http.server 8080
或者前端网页直接部署
```

## 📝 词汇数据格式

每个词条包含以下字段：

```javascript
{
  word: "abandon",           // 单词
  phonetic: "/əˈbændən/",    // 音标
  meaning: "v. 放弃，抛弃",   // 中文释义
  example: "He abandoned his car...",  // 例句
  frequency: 5,              // 频率等级 (1-5, 5为最高频)
  synGroup: "give_up"        // 同义词组 ID (null 表示无分组)
}
```
